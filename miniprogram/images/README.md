# TabBar 图标说明

小程序 tabBar 需要以下 6 个图标文件（81x81 像素，PNG 格式）：

| 文件名 | 用途 |
|--------|------|
| tab-schedule.png | 课程表 tab（未选中） |
| tab-schedule-active.png | 课程表 tab（选中） |
| tab-group.png | 家庭组 tab（未选中） |
| tab-group-active.png | 家庭组 tab（选中） |
| tab-profile.png | 我的 tab（未选中） |
| tab-profile-active.png | 我的 tab（选中） |
| default-avatar.png | 默认头像 |

## 获取图标

推荐使用以下免费图标网站：
- [iconfont](https://www.iconfont.cn/) - 阿里巴巴图标库
- [IconPark](https://iconpark.oceanengine.com/) - 字节跳动图标库

下载 81x81 像素的 PNG 图标，未选中状态用灰色(#999999)，选中状态用主题色(#4A90D9)。

## 临时方案

在微信开发者工具中，如果图标文件不存在，tabBar 会显示纯文字，不影响功能使用。
可以先开发调试，后续再补充图标。
